diskget.c
Retrieves a specified file from the disk image.

usage
./diskget <disk_image> <filename>

diskput.c
Puts a specified file into the disk image.

usage
./diskput <disk_image> </path/to/filename>

diskinfo.c
Provides detailed information about the disk image.

usage
./diskinfo <disk_image>


disklist.c
lists contents of disk

usage
./disklist <disk_image>



Compilation
Type 'make' into terminal
